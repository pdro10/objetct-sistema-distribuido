1. Ativar o ambiente

conda activate omni-env

2. Gerar o cÓdigo da IDL

omniidl -bpython soma.idl

3. Iniciar o OmniNames (Serviço de Nomes)

Execute em um terminal separado ou em background:

omniNames -start

4. Iniciar o servidor

Execute em outro terminal:

python server.py

5. Iniciar o Cliente

Execute em outro terminal:

python client.py
